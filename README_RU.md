## О проекте

**AI Aggregator** — это современная веб-платформа для унифицированной работы с несколькими AI-провайдерами через единый интерфейс. Проект объединяет ведущие языковые модели (Z.ai, OpenAI, Anthropic, DeepSeek) с возможностью сравнения ответов, генерации изображений, веб-поиска и гибкой настройки. Платформа предназначена для разработчиков, исследователей и всех, кто регулярно использует AI-модели в работе и хочет иметь единую точку доступа.

Все материалы представлены на русском языке.

## Ключевые возможности

- **4 AI-провайдера и 11 моделей** — Z.ai (GLM-4 Plus/Flash/Air/Long), OpenAI (GPT-4o, GPT-4o Mini), Anthropic (Claude Sonnet 4, Claude 3.5 Haiku), DeepSeek (V3, R1)
- **Режим сравнения** — отправка одного запроса двум моделям одновременно с разделённым просмотром ответов
- **Генерация изображений** — встроенный инструмент для создания изображений через AI
- **Веб-поиск** — поиск информации в интернете прямо из интерфейса чата
- **Умная рекомендация модели** — автоматический выбор оптимальной модели на основе контекста запроса
- **Шаблоны промптов** — сохранение и быстрый доступ к часто используемым запросам
- **Управление диалогами** — создание, поиск, тегирование, закрепление и экспорт в Markdown
- **Тёмная/светлая тема** — переключение с сохранением состояния
- **Горячие клавиши** — быстрые команды для всех основных действий
- **Статистика использования** — мониторинг токенов по провайдерам и моделям
- **Система уведомлений** — звуковые оповещения о завершении ответа
- **Адаптивный интерфейс** — полностью отзывчивый дизайн для мобильных устройств, планшетов и десктопов
- **Анимации и переходы** — плавный интерфейс на базе Framer Motion
- **Персистентность** — сохранение диалогов и настроек в localStorage

## Установка и запуск

### Предварительные требования

- **Node.js** версии 18.17 или выше (рекомендуется 20+)
- **Bun**, **npm**, **yarn** или **pnpm** в качестве пакетного менеджера

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/dupleymi-aup/ai-aggregator.git
cd ai-aggregator

# Установить зависимости
npm install

# Настроить переменные окружения
cp .env.example .env.local

# Запустить в режиме разработки
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

### Сборка для продакшена

```bash
npm run build
npm start
```
