## О проекте

**AI Aggregator** — современная веб-платформа для унифицированной работы с несколькими AI-провайдерами через единый интерфейс. Поддерживает Z.ai, OpenAI, Anthropic и DeepSeek с возможностью сравнения ответов, генерации изображений и веб-поиска.

Полные описания доступны на русском и английском языках:
- [README на русском языке](README_RU.md)
- [README in English](README_EN.md)

### Ключевые возможности

- **4 AI-провайдера и 11 моделей** — Z.ai, OpenAI, Anthropic, DeepSeek
- **Режим сравнения** — один запрос двум моделям одновременно
- **Генерация изображений** — встроенный AI-генератор
- **Веб-поиск** — поиск информации прямо из чата
- **Умная рекомендация** — автоматический выбор модели под задачу
- **Шаблоны промптов** — сохранение часто используемых запросов
- **Управление диалогами** — поиск, теги, закрепление, экспорт в Markdown
- **Статистика** — мониторинг токенов по провайдерам и моделям
- **Тёмная/светлая тема** и **горячие клавиши**

### Поддерживаемые провайдеры

| Провайдер | Модели |
|-----------|--------|
| **Z.ai** 🟣 | GLM-4 Plus, GLM-4 Flash, GLM-4 Air, GLM-4 Long |
| **OpenAI** 🟢 | GPT-4o, GPT-4o Mini |
| **Anthropic** 🟠 | Claude Sonnet 4, Claude 3.5 Haiku |
| **DeepSeek** 🔵 | DeepSeek V3, DeepSeek R1 |

### Быстрый старт

```bash
git clone https://github.com/dupleymi-aup/ai-aggregator.git
cd ai-aggregator
npm install
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)
